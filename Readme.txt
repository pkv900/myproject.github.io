About: This is a Online Shopping Website INDIASHOP

This project is developed on Visual Studio Code and tested on Xammp local server.
It uses web development technologies like HTML5, CSS3 and BOOTSTRAP.

Prerequisites:
1. Xammp server
2. Web Browser such as Google Chrome, Microsoft Edge, Mozilla Firefox etc.

How to run:
1.Copy the directory to a new folder in xammp/htdocs.
2. Rename the new folder as myproject.
3. Turn on Apache and MySQL on XAMMP Server.
4. Create database in phpmyadmin and import from mydb databse present in this folder. 
5. Type on browser localhost/spotlight and press Enter.
6. Admin id-admin
   Password-admin123
7.User Can Create New Id.
